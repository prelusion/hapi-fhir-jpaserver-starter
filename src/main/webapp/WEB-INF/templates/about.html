About This Server

This server provides a complete implementation of the FHIR specification for Wild Sea.

This server is build upon a kubernetes cluster which is configured through the help of the development process owned by Wild Sea.

Data On This Server

The data of this server is saved through the database scheme given by HAPI FHIR. See the architectural document for an indepth view of the database configuration
